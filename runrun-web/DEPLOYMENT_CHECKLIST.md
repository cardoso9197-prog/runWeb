## Deployment to Netlify

### Option 1: Netlify UI (Recommended)
- [ ] Login to https://netlify.com
- [ ] Click "Add new site" → "Import an existing project"
- [ ] Connect Git repository (GitHub/GitLab/Bitbucket)
- [ ] Configure build settings:
  - Build command: `npm run build`
  - Publish directory: `.next`
  - Base directory: `runrun-web` (if monorepo)
- [ ] Add environment variable:
  - Key: `NEXT_PUBLIC_API_URL`
  - Value: Your Railway backend URL
- [ ] Click "Deploy site"
- [ ] Wait for deployment to complete

### Option 2: Netlify CLI
- [ ] Install CLI: `npm install -g netlify-cli`
- [ ] Login: `netlify login`
- [ ] Navigate to runrun-web: `cd runrun-web`
- [ ] Build: `npm run build`
- [ ] Deploy: `netlify deploy --prod`

## Troubleshooting

### Issue: Build fails on Netlify
**Check:**
- Node version compatibility (should be 18+)
- All dependencies in package.json
- Environment variables set correctly

### Issue: Admin dashboard shows API error
**Check:**
- Railway backend is running
- NEXT_PUBLIC_API_URL is correct in Netlify env vars
- CORS enabled on backend for Netlify domain
- Admin key header matches backend

### Issue: Pages show 404 on refresh
**Check:**
- netlify.toml is in repository
- Redirect rules are configured
- Next.js plugin is active
